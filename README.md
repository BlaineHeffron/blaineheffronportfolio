# Blaine Heffron Portfolio

Static portfolio site for <https://blaineheffron.com>.

## Structure

- `index.html` - main portfolio and consulting page
- `research.html` - research background page, also available at `/research`
- `card/` - compact profile card page
- `css/` - site styles
- `js/` - site behavior
- `images/` - portrait images
- `assets/media/` - social sharing and icon assets
- `static/` - favicons and downloadable PDFs

## Deployment

The site is deployed as static files. There is no build step.

Netlify uses `netlify.toml` and publishes the repository root. GitHub Pages uses `.github/workflows/publish.yaml` and also uploads the repository root.
